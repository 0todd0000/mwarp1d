Overview¶

This notebook provides a brief overview of the purpose and functionality of mwarp1d using the following three repository scripts, which are run inline in this notebook.

  1. fig_dataset.py
  2. fig_landmarks.py
  3. fig_warp_overview.py

Running this notebook and/or the scripts will test all main mwarp1d functionality.

For detailed use instructions, refer to mwarp1d's online documentation.

Summary¶

mwarp1d allows users to nonlinearly register 1D data using either (1) landmark warping or (2) manual warping. This can be achieved via scripting, as demonstrated above, or using the mwarp1d GUI, as described in a series of screencasts.
